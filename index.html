Latitude vs. Weather Factor

This project was to analyze how various aspects of the weather changed with global latitudes on 01/05/2017. Weather data was pulled from the OpenWeatherMap API for over 500 cities.

Matplotlib was used to scatter plot the latitude of each city vs. various aspects of the weather (temperature, cloudiness, wind speed, and humidity).

This site provides visualizations, comparative analysis, and source data.
